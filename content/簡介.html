Git 與 Github

Git-是一個版本控制系統，可記錄所有狀態變化和隨時切換至某個版本，也能清楚更新版本時修改了什麼內容，Git還有一個優點 分散式系統->即使在無網路和無伺服器的環境下依舊可以用Git進行版控，等有網路在同步就好。

Github->一種網頁型態，可託管各種Git可進行分支和共享成果，主要的功能還是社群網路功能，能讓人便利的分享原始碼，當你有Follow的人創分支的時能夠即時追蹤，以便觀察開源軟體社群中的參與。
